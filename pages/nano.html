nano Topics

My editor of choice for doing most work is vscode or sublime. I will occasionally use nano to make quick edits to files, especially when they are not part of larger projects. Since I use it rarely, I don't remember most commands. I made myself a list of the commands I keep having to look up.

Viewing Text

  • nano -v filename.txt Open file as read-only
  • Esc then $ Toggle soft word wrap
  • Esc then N Toggle line numbers

Select, cut, and paste text

  • Ctrl Shift 6 Set start point for text selection, then move to end point
  • Ctrl K Cut selected text (or whole line if no text selected)
  • Ctrl U Paste selected text

By default, nano is presented in monochome. The look and feel can be customized by adding the following lines to your ~/.nanorc file. The colors can be whatever you want. When specifying more than one color, do not incude a space with the comma. View more options in the nano documetation.

  • set softwrap Keeps softwrap always on, so you don't have to Escape-$
  • set numbercolor yellow Sets the color of the line numbers
  • set titlecolor white,red Sets the color of the file titlebar as foreground,background
  • set functioncolor cyan and set keycolor green Sets the color of the keystrokes and functions displayed at the bottom of the screen
  • include /usr/local/share/nano/*.nanorc Tells nano to include language-specific syntax highlighting, as defined for all the major languages in that folder.
