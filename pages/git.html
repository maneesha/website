Git Topics

Because I'm always forgetting certain commands in Git and GitHub, I made myself a list of the commands I keep having to look up.

This gives a summary of git log:

git log --oneline

Like git log but also shows the files that were changed in each commit.

git log --name-status

Show the file diff in the log. It's like getting git log and git diff all at once.

git log -p

Show all commits for a specific file.

git log -p -- _config.yml

Show all commits in a date range.

git log --after="2019-07-01" --before="2019-09-30"

Show all commit where messages contain a term.

git log --grep="mesa verde"

Show all commits where that term was changed in a file itself. Use the -S flag for text search; -G for regex search

git log -S"heisenberg" 
# OR 
git log -G"\bh[a-z]*rg\b"
