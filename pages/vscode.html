Wrapping text in html tags

If you want to wrap text in html tags but don't want to type the opening and closing tag for each one, use Emmet Wrap with Abbreviation. Select the text to be wrapped and hit Ctrl or Cmd Shift P. Start typing "Emmet..." and autocomplete will take you to Emmet Wrap with Abbreviation. Hit Enter to choose this and then type in the tag you want to use without pointy braces. Your text will now be wrapped in whatever tags you enter (regardless of whether it is a valid tag).
