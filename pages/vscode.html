VS Code Topics

Because I'm always forgetting certain commands in VS Code, I made myself a list of the commands I keep having to look up.

Autocomplete html tags

If you are in a html file, typing the tag (without the pointy braces) and then hitting Tab will autocomplete the opening and closing tags, including the pointy braces.

Wrapping text in html tags

If you want to wrap text in html tags but don't want to type the opening and closing tag for each one, use Emmet Wrap with Abbreviation. Select the text to be wrapped and hit Ctrl or Cmd Shift P. Start typing "Emmet..." and autocomplete will take you to Emmet Wrap with Abbreviation. Hit Enter to choose this and then type in the tag you want to use without pointy braces. Your text will now be wrapped in whatever tags you enter (regardless of whether it is a valid tag).

Previewing Markdown

To preview raw Markdown as rendered Markdown in a new tab in the same pane as your raw file: Hit Ctrl or Cmd Shift V.

To preview raw and rendered Markdown side by side: Hit Ctrl or Cmd K, then release and hit V.

VS Code can give you alerts about errors and other status notices.

Error counts

Next to the file name in the Explorer or on the tab of an open file, you may see a number. This is the count of errors in your file. This can include style or syntax errors. Additional information is available from the View >> Problems menu (shortcut Ctrl or Cmd Shift M.)

File modification

If you use VS Code with Git, you may see green or blue lines next to your line numbers. Green means that these are new lines since your last commit; blue means these are modified lines since your next commit.

View unsaved changes

When you have unsaved changes in your file, the file name will be displayed with a dot in the editor tab and in the "Open Editors" list. To see a diff of what's been changed, do one of the following. Either will give you the same diff view.

  • Right click on the file in the "Open Editors" view and select "Compare with saved"
  • Hit Ctrl or Cmd K, then release and hit D
